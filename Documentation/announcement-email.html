Subject: Introducing Agent Forge -- Your GitHub Command Center, Inside Your Editor

Hi there,

What if you never had to open GitHub in your browser again?

I'm excited to share Agent Forge -- an open-source collection of AI agents and prompt commands that turn VS Code's Copilot Chat into a complete GitHub command center.

https://github.com/accesswatch/agent-forge

What Is Agent Forge?

Agent Forge is a curated set of 5 specialized AI agents and 28 slash commands that work natively with GitHub Copilot. They live inside your editor and talk to GitHub for you -- finding issues, reviewing code, tracking team progress, managing releases, monitoring security, and even posting comments -- all without opening a single browser tab.

Type a question in plain English. Get back organized, prioritized, actionable answers.

What Can It Do?

Here's a taste:

  • @daily-briefing morning briefing -- Sweeps every repo you touch and builds a prioritized dashboard with issues, PRs, releases, CI status, security alerts, and community reactions. In 30 seconds. Every morning.

  • /review-pr owner/repo#123 -- Full code review with risk assessment, before/after snapshots, CI results, security flags, and a working checklist. Leave line-specific comments and code suggestions without leaving chat.

  • /triage -- Smart priority scoring for your issue backlog with community sentiment, release context, and batch reply capabilities.

  • /merge-pr, /draft-release, /sprint-review, /security-dashboard -- The full lifecycle, from commit to ship.

  • 28 slash commands total -- everything from /my-issues to /manage-branches to /notifications.

The Agents

Agent What It Does
Daily Briefing Prioritized overview of everything happening across all your repos
Issue Tracker Find, triage, reply to, and manage issues with smart search and templates
PR Review Full code reviews with risk assessment, inline commenting, and merge
Analytics Team velocity, bottlenecks, code hotspots, and workload insights
A11y Tracker Accessibility change monitoring with WCAG/ARIA cross-references

Zero Setup Required

If GitHub Copilot works in your VS Code, Agent Forge works. No API keys. No CLI tools. No extensions beyond what you already have. Clone the repo, open it as a workspace, and type @daily-briefing morning briefing. That's it.

Accessibility First

Every document the agents generate comes in dual format -- Markdown for editing and HTML optimized for screen readers. Skip links, ARIA landmarks, accessible tables, descriptive link text, WCAG AA contrast compliance, reduced-motion support. Accessibility isn't a feature -- it's foundational.

We Want Your Contributions

Agent Forge is open source (MIT) and we'd love your help making it better:

  • Submit a new agent -- Have a workflow you've automated? Share it with the community.
  • Add a prompt command -- Got a common task that deserves a slash command? Build it.
  • Improve documentation -- Clearer guides, better examples, new integration patterns.
  • Report bugs -- Found something that doesn't work right? Open an issue.
  • Star the repo -- It helps others discover the project.

We use a fork-and-PR workflow with branch protection on main. Every contribution gets a thoughtful review within 48 hours. See our Contributing Guide for the full details.

Works With Other AI Platforms

Agent Forge is built for GitHub Copilot, but the agents and prompts can be adapted for Claude AI, ChatGPT, and OpenAI API integrations. Platform-specific guides are included.
